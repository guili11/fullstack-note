# 请求封装（axios）

## 一、目标

统一前端所有 HTTP 请求，解决：

- baseURL / timeout 统一配置
- token 自动注入
- 响应统一解析
- 错误统一处理（401 / 500 / 业务错误）
- 避免组件中直接写请求细节

***

## 二、axios调用封装（对外 API）  --放在services层中

```ts
import request from '../utils/request'
import type { LoginForm ,LoginResponse } from '../types/user'


//类型的使用，就是在函数返回的时候，指定返回值的类型，调用的时候，直接自动判断类型了
export async function launchLogin(values:LoginForm):Promise<LoginResponse>{
  // 这里的返回值类型还真没有code和msg，因为在拦截器中已经处理了
  const res = await request.post('/api/auth/login',values) 
  return res as unknown as LoginResponse
}
```

## 三、核心实现（request.ts）

```ts
// src/utils/request.ts
import axios from 'axios'
import { useUserStore } from '@/store/userStore'

// 创建实例
const instance = axios.create({
  baseURL: '/api',
  timeout: 10000,
})

// 请求拦截器（注入 token）
instance.interceptors.request.use(
  (config) => {
    const { token } = useUserStore.getState()

    if (token) {
      config.headers.Authorization = `Bearer ${token}`
    }

    return config
  },
  (error) => Promise.reject(error)
)

// 响应拦截器（统一处理）
instance.interceptors.response.use(
  (response) => {
    const res = response.data

    if (res.code !== 0) {
      return Promise.reject(res)
    }

    return res.data
  },
  (error) => {
    const status = error.response?.status

    if (status === 401) {
      const { logout } = useUserStore.getState()
      logout()
      window.location.href = '/login'
    }

    return Promise.reject(error)
  }
)

export default instance
```

## 四、设计约定（非常重要）

### 1️⃣ 统一返回结构

后端返回格式建议：

```json
{
  "code": 0,
  "message": "ok",
  "data": {}
}
```

***

### 2️⃣ 成功 / 失败规则

```
code === 0 → 成功
code !== 0 → 业务错误
```

***

### 3️⃣ 请求路径规范

```
统一使用 /api 前缀（配合 Vite proxy）
```

***

## 五、常见问题

### ❌ 组件里直接写 axios

```ts
axios.get('/api/users') ❌
```

应该：

```ts
request.get('/users') ✅
```

***

### ❌ token 写死

```ts
headers: { Authorization: 'xxx' } ❌
```

应该走拦截器自动注入

***

### ❌ 每个接口都 try/catch

应该：

- 拦截器先处理泛型错误 401  http网络  业务状态码错误（code !== 0）→ 抛出错误
- 请求层只负责"抛错"
- controller 层兜底处理

***

## 六、职责边界

```
拦截器：处理通用错误（HTTP + 业务状态码）
   ↓
抛出错误
   ↓
Controller：统一 catch，显示提示，记录日志
   ↓
组件：只负责 UI，不处理业务逻辑
```

***

## 七、一句话总结

```
请求封装 = axios 实例 + 拦截器 + 统一规范
```

## 八、完整示例（带类型）

### 1️⃣ 类型定义

```ts
// types/user.ts
export interface User {
  id: number
  name: string
  email: string
  createdAt: string
}

export interface UserListParams {
  page: number
  size: number
}

export interface CreateUserParams {
  name: string
  email: string
  password?: string
}
```

***

### 2️⃣ Service 层

```ts
// services/user.ts
import request from '@/utils/request'
import type { User, UserListParams, CreateUserParams } from '@/types/user'

export function getUserList(params: UserListParams) {
  return request<User[]>({
    url: '/users',
    method: 'GET',
    params
  })
}

export function createUser(data: CreateUserParams) {
  return request<User>({
    url: '/users',
    method: 'POST',
    data
  })
}

export function updateUser(id: number, data: Partial<User>) {
  return request<User>({
    url: `/users/${id}`,
    method: 'PUT',
    data
  })
}

export function deleteUser(id: number) {
  return request({
    url: `/users/${id}`,
    method: 'DELETE'
  })
}
```

***

### 3️⃣ Controller 层

```ts
// controllers/user.ts
import type { User, UserListParams, CreateUserParams } from '@/types/user'
import { getUserList, createUser, updateUser, deleteUser } from '@/services/user'

export async function fetchUsers(page = 1, size = 10) {
  return getUserList({ page, size })
}

export async function createUserAction(userData: CreateUserParams) {
  return createUser(userData)
}

export async function updateUserAction(id: number, userData: Partial<User>) {
  return updateUser(id, userData)
}

export async function deleteUserAction(id: number) {
  return deleteUser(id)
}
```

***

### 4️⃣ 组件层

```tsx
// components/UserList.tsx
import { useEffect, useState } from 'react'
import { fetchUsers, createUserAction } from '@/controllers/user'
import type { User } from '@/types/user'

export function UserList() {
  const [users, setUsers] = useState<User[]>([])
  const [loading, setLoading] = useState(false)

  useEffect(() => {
    const loadUsers = async () => {
      setLoading(true)
      try {
        const data = await fetchUsers()
        setUsers(data)
      } catch (error) {
        console.error('加载失败')
      } finally {
        setLoading(false)
      }
    }
    loadUsers()
  }, [])

  const handleCreate = async () => {
    try {
      const newUser = await createUserAction({
        name: '张三',
        email: 'zhangsan@example.com'
      })
      setUsers((prev) => [...prev, newUser])
    } catch (error) {
      console.error('创建失败')
    }
  }

  if (loading) return <div>加载中...</div>

  return (
    <div>
      <button onClick={handleCreate}>创建用户</button>
      {users.map((user) => (
        <div key={user.id}>{user.name}</div>
      ))}
    </div>
  )
}
```
