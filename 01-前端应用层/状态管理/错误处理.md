# ✅ 五、错误处理（四层版）

***

## ❗规则：

```txt
service：抛错误
controller：处理错误（核心）
view：只展示（controller控制）
store：不碰错误
```

***

## 实现：

### service

```ts
//其实不try -catch 就是自动抛出了
throw new Error('创建失败')
```

***

### controller

```ts
catch (e) {
  handleError(e)
}
```

***

### utils/error.ts

```ts
export function handleError(error) {
  console.error(error)
  message.error(error.message || '请求失败')
}
```
