# React 响应式原理

## 核心流程

```
1️⃣ useState 创建状态
       ↓
2️⃣ 组件渲染（读取状态）
       ↓
3️⃣ 用户操作（如点击）
       ↓
4️⃣ 调用 setXxx 更新状态
       ↓
5️⃣ React 检测到状态变化
       ↓
6️⃣ 重新渲染组件（对比 Virtual DOM）
       ↓
7️⃣ 更新真实 DOM
```

---

## 一句话理解

```
状态变化 → 触发重新渲染 → 更新 UI
```

---

## 关键点

- **状态是唯一的"数据源"**
- **setXxx 是唯一更新方式**
- **组件 = f(state)**：UI 由状态决定
